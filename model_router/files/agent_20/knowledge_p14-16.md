# 计算机软件
下面介绍尼康的主要计算机软件应用程序。有关最新信息，请访问尼康网站。
- 尼康计算机软件可从尼康下载中心获取。请查看版本和系统要求，并确保下载最新版本。
  https://downloadcenter.nikonimglib.com/

## 尼康工坊
查看和编辑使用尼康数码照相机拍摄的照片。

![尼康工坊图标](assets/knowledge_p14-16_001.png)

- 使用尼康工坊的Nikon Transfer 2组件将照片传送至计算机。
- 可以使用尼康工坊将尼康独特的NEF/NRW（RAW）照片转换成其他图像格式（NEF/RAW处理）。
- 可以使用尼康工坊调整使用尼康数码照相机所拍摄照片的亮度和对比度。
- 可以使用尼康工坊调整和管理自定义优化校准。
- 确保下载最新版本，因为早期版本可能无法从照相机下载照片或打开NEF（RAW）照片。
- 尼康工坊软件可从尼康下载中心进行下载。
  https://downloadcenter.nikonimglib.com/zh-cn/products/564/NX_Studio.html

## 尼康远控
用于照相机连接到计算机时的联机拍摄。曝光、白平衡和其他照相机设定可以使用计算机显示屏中的控制进行远程调整。

![尼康远控图标](assets/knowledge_p14-16_002.png)

## Wireless Transmitter Utility
若要将照相机连接至网络，需要Wireless Transmitter Utility。为照相机和计算机配对，并通过无线网络下载照片。

![Wireless Transmitter Utility图标](assets/knowledge_p14-16_003.png)
