# 触控控制
触摸感应显示屏具备各种控制，您可通过用手指触摸屏幕来操作这些控制。在取景器拍摄期间，触控控制被禁用且无法用于对焦等操作。

![触控操作示意图](assets/knowledge_p36-39_001.png)

## 对焦并释放快门
- 轻触显示屏可对焦于所选点（触控AF）。

![触控AF操作示意图](assets/knowledge_p36-39_002.png)
- 在照片模式下，快门将在您从屏幕中抬起手指时释放（触控快门）。
- 触控快门/AF设定可通过轻触<svg width="14" height="16" viewBox="0 0 14 16" fill="none" xmlns="http://www.w3.org/2000/svg"><path d="M13 14V2H1V14H13Z" stroke="currentColor" stroke-width="2"/><path d="M9 5L5 8L9 11V5Z" fill="currentColor"/></svg>图标进行调整（<svg width="16" height="12" viewBox="0 0 16 12" fill="none" xmlns="http://www.w3.org/2000/svg"><path d="M1 1H15V7H1V1Z" stroke="currentColor" stroke-width="2"/><path d="M4 11H12" stroke="currentColor" stroke-width="2" stroke-linecap="round"/></svg> 132）。

![触控快门设置位置示意图](assets/knowledge_p36-39_003.png)

## 调整设定
- 轻触屏幕中加亮显示的设定。

![加亮设定选择示意图](assets/knowledge_p36-39_004.png)
- 随后即可通过轻触图标或滑块选择所需选项。
- 轻触<svg width="14" height="14" viewBox="0 0 14 14" fill="none" xmlns="http://www.w3.org/2000/svg"><path d="M11 7H3M3 7L7 11M3 7L7 3" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"/></svg>或按下<kbd>OK</kbd>可选择所选项并返回上一级显示。

![设置调整界面示意图](assets/knowledge_p36-39_005.png)

## 播放
- 在全屏播放过程中向左或向右轻拨可查看其他照片。

![播放滑动切换示意图](assets/knowledge_p36-39_006.png)

照相机控制

<!-- page 38 -->
- 在全屏播放中，轻触屏幕底部会显示一个图像快进条。在快进条上向左或向右滑动手指可快速滚动至其他照片。

![快进条操作示意图](assets/knowledge_p36-39_007.png)
- 若要放大在全屏播放期间显示的照片，请使用分开动作或者快速轻触屏幕2次。您可通过使用手指分开动作进行放大和并拢动作进行缩小来调整缩放率。

![缩放操作示意图](assets/knowledge_p36-39_008.png)
- 缩放期间使用滑动动作可查看图像的其他部分。
- 缩放期间快速轻触屏幕2次可取消缩放。
- 在全屏播放过程中使用并拢动作可“缩小”至缩略图查看。使用并拢和分开动作可从4张、9张及72张中选择图像显示数量。

![缩略图切换示意图](assets/knowledge_p36-39_009.png)
