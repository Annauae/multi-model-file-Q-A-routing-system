# 🔍和🔍❏（？）按钮
在拍摄或播放期间放大或缩小显示屏中的显示。

![放大/缩小按钮示意图](assets/knowledge_p44-45_001.png)

## ■■ 拍摄
按下🔍可在拍摄期间放大镜头视野。按下🔍可增加缩放率，按下🔍❏（？）则可减小缩放率。

## ■■ 播放
按下🔍可在全屏播放期间放大照片。按下🔍可增加缩放率，按下🔍❏（？）则可减小缩放率。当照片全屏显示时，按下🔍❏（？）可“缩小”至缩略图列表。当显示72张画面时，按下🔍❏（？）可选择日历播放。

# AE-L/AF-L（<sub>O<sub>m</sub></sub>）按钮
