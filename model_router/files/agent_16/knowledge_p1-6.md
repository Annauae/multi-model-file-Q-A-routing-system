# 安全须知

为预防对您或他人造成人身伤害或财产损失，请在使用前仔细阅读“安全须知”，并以正确的方法使用。请在阅读本使用说明之后妥善保存，以便随时查阅。

⚠️ **危险**：表示“极有可能造成人员死亡或负重伤的内容”。
⚠️ **警告**：表示“有可能造成人员死亡或负重伤的内容”。
⚠️ **注意**：表示“有可能造成人员负轻伤或财产损失的内容”。

本节使用以下图示和符号对必须遵守的内容作以分类。

| 符号 | 含义 |
| --- | --- |
| ⭕ | 表示不允许进行的行为。 |
| ⭕（带扳手） | 表示不得自行进行拆解、修理或改装。 |
| ⭕（带手掌） | 表示紧贴着人体或物体时不允许进行的行为。 |
| ⚫（带感叹号） | 表示必须进行的行为。 |
