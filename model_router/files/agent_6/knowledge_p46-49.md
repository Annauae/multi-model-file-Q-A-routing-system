# ▌▌使用菜单
您可使用多重选择器和OK按钮操作菜单。

![多重选择器操作图示](assets/knowledge_p46-49_003.png)

| 序号 | 功能 |
| ---- | ---- |
| 1 | 向上移动光标 |
| 2 | 选择加亮显示的项目 |
| 3 | 显示子菜单，选择加亮显示的项目或向右移动光标 |
| 4 | 向下移动光标 |
| 5 | 取消并返回上一级菜单，或向左移动光标 |

---
