## 打开显示屏

慢慢旋转显示屏，不要过度用力。

| 编号 | 部件 | 参见 |
| :--- | :--- | :--- |
| ① | 从相机背面开始，将显示屏向右旋转打开 | 第14页图1 |
| ② | 将显示屏继续向左旋转，直至完全展开 | 第14页图2 |
| ③ | 将显示屏旋转至所需角度，例如竖直或水平方向 | 第14页图3 |

> **注意**：操作时请轻柔缓慢，避免施加过大的力量，以免损坏显示屏机构。
