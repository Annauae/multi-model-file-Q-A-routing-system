# 控制面板
照相机开启时，控制面板也会开启。
光圈（0 154、0 155）在控制面板中显示。

![控制面板光圈显示示例](assets/knowledge_p20-21_002.png)

- 如果将光圈设为F0.95，则控制面板中将显示[.95]。
- 若将[扩展快门速度范围(M)]设为[开启]，当以60秒或更长时间的快门速度开始拍摄时，显示会切换以指示快门已打开。
- 根据镜头安装方式的不同，可能会显示警告[--]（0 593）。
- 当连接到计算机或智能手机时，可能会显示[PC]以指示连接正在进行中。
