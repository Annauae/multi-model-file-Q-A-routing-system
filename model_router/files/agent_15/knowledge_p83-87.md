# 查看照片
按下 ▶ 按钮可在显示屏或取景器中查看使用照相机所记录的照片和视频。

![相机播放按钮图示](assets/knowledge_p83-87_001.png)

- 按下 ◀ 或 ▶，或在显示屏上向左或向右轻拨可查看其他图像。

![查看照片界面图示](assets/knowledge_p83-87_002.png)

- 视频由显示屏左上方的 🎞 图标标识。按下 OK 按钮，或轻触显示屏中的 ▶ 图标可播放视频。

![视频播放预览图示](assets/knowledge_p83-87_003.png)

- 若要结束播放并返回拍摄模式，请半按快门释放按钮。

# 删除不想要的照片
您可如下所述删除照片。**请注意，照片一旦被删除，将不能恢复。**

## 1 显示照片。

![相机播放按钮图示](assets/knowledge_p83-87_005.png)

- 按下 ▶ 按钮开始播放，然后按下 ◀ 或 ▶ 直至显示所需照片。

查看照片
118

<!-- page 87 -->

## 2 删除照片。

![相机删除按钮图示](assets/knowledge_p83-87_006.png)

- 按下 🗑 按钮；屏幕中将显示一个确认对话框。再次按下 🗑 按钮将删除图像并返回播放。
- 若要不删除照片直接退出，请按下 ▶。

![删除确认界面图示](assets/knowledge_p83-87_007.png)

---

✏️ 播放菜单[删除]项目

- 删除在所选日期拍摄的照片
- 删除所选文件夹中的所有照片
